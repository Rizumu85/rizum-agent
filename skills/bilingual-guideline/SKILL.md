# Bilingual Guideline

Once activated, this guideline stays active for the entire conversation.

## Communicate Bilingually

**English in files. Chinese in summaries.**

- Write project files, code, comments, and technical docs in English.
- Summarize chat changes in Chinese.
- Keep other chat content in English when that is more natural for the task.
